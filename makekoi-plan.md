# みつ越し商店の奇跡 〜負けてよガンマ、恋のために〜

## 1. 企画概要

### 1.1 タイトル
みつ越し商店の奇跡 〜負けてよガンマ、恋のために〜

### 1.2 ジャンル
土下座と値切りのクソゲーアクションRPG（1プレイ短時間・笑い特化）

### 1.3 コンセプト
デートのために恋人へ服をフルセット買いたい貧乏主人公が、頑固店長「ガンマ」を相手に値切り勝負を挑む。プレイヤーはミニゲームで必死に値切り、勝てば安く高ランク品を獲得、負ければ服ランクが下がる。最終コーデのチグハグ具合がエンディングの笑いに直結する。

### 1.4 想定プレイ時間
1周 5〜8分

### 1.5 想定プラットフォーム
PC（キーボード操作）

## 2. ターゲット

### 2.1 想定ユーザー
1. 短時間で笑えるゲームを遊びたい人
2. 配信映えするバカゲーを探している人
3. リトライしてネタエンディングを集めたい人

### 2.2 体験価値
1. ギリギリ成功する緊張感
2. 失敗しても笑える安心感
3. 結果画面で爆笑できるオチ

## 3. ゲーム全体フロー

1. タイトル
2. デート相手の性別選択
3. デート相手のランダム要望決定
4. 買い物フェーズ（全7部位）
5. 最終コーデ評価
6. エンディング
7. リザルト（再挑戦）

## 4. コアループ

1. 部位を提示（例: 靴）
2. 現在提示ランクは「おしゃれ」から開始
3. 30秒の値切りバトル実施
4. 成功: そのランクで購入確定
5. 失敗1回目: ランクを1段階ダウンして再交渉
6. 失敗2回目: ランクをさらに1段階ダウンして強制購入
7. 次部位へ進行

## 5. 部位とランク設計

### 5.1 購入部位（全7）
1. 上着
2. ズボン
3. 靴
4. アクセサリー
5. バッグ
6. ヘア小物

### 5.2 服ランク
1. おしゃれ（最高）
2. 普通
3. ダサい平凡（最低）

### 5.3 価格帯（仮）
1. おしゃれ: 12000円
2. 普通: 7000円
3. ダサい平凡: 3000円

注記: 実装では整数ポイントを使い、表示だけ円表記にする。

## 6. キャラ選択と要望システム

### 6.1 デート相手性別
1. 女性
2. 男性
3. 指定なし（ランダム）

### 6.2 ランダム要望（例）
1. 今回は下着をこだわりたい
2. 靴だけは本気で頼む
3. 上着がダサいと泣く
4. 全体の色味を統一してほしい

### 6.3 要望の効果
要望対象部位が「おしゃれ」なら評価ボーナス、
「普通」は補正なし、
「ダサい平凡」は評価ペナルティ。

## 7. 値切りバトル仕様（ターン制アクションミニゲーム）

### 7.1 1バトルの制限時間
30秒

### 7.2 アクション種別
1. 土下座ジャスト
2. 財布アピール連打
3. 言い訳カード選択

### 7.3 アクション詳細

#### 7.3.1 土下座ジャスト
1. ゲージが表示される
2. 成功ゾーンでボタン入力すると成功
3. 成功で「値切り力」大幅上昇
4. 失敗で「ガンマ機嫌」悪化

#### 7.3.2 財布アピール連打
1. 指定時間内に連打回数を稼ぐ
2. 規定回数到達で成功
3. 成功で「値切り力」中上昇

#### 7.3.3 言い訳カード選択
1. 3択から台詞を選んで、がんばとジャンげん
2. 正解で「ガンマ機嫌」改善
3. 一定時間内に時間が経つことによって商品のランクが下がり、ジャンゲンでもしかったらバーが増える

### 7.4 バトル勝敗判定
制限時間終了時に以下を比較する。

1. 値切り力 >= 目標値: 交渉成功
2. 値切り力 < 目標値: 交渉失敗

## 8. パラメータ設計

### 8.1 主パラメータ
1. 値切り力（NegotiationPower）
2. ガンマ機嫌（GammaMood）
3. プレッシャー（Pressure）

### 8.2 初期値（仮）
1. 値切り力: 0
2. ガンマ機嫌: 50
3. プレッシャー: 0

### 8.3 目標値の計算（仮）
目標値 = 部位難易度 + ランク補正 + 終盤補正

部位難易度は部位ごとの固定値、
ランク補正はおしゃれほど高い、
終盤補正は6部位目以降で加算。

## 9. 難易度カーブ

1. 1〜2部位目: 成功しやすい
2. 3〜5部位目: 標準
3. 6〜7部位目: 失敗リスク高め

狙い: 前半で気持ちよく成功させ、後半でチグハグ発生率を上げて笑いを作る。

## 10. デート評価仕様

### 10.1 スコア計算
総合スコア = 部位スコア合計 + 要望ボーナス + 統一感ボーナス - チグハグペナルティ

### 10.2 部位スコア（仮）
1. おしゃれ: +15
2. 普通: +8
3. ダサい平凡: +2

### 10.3 チグハグ判定（仮）
以下のいずれかでチグハグ扱い。

1. 上半身平均ランクと下半身平均ランクの差が2以上
2. 「おしゃれ」と「ダサい平凡」が同時に3部位以上

### 10.4 エンディング分岐（仮）
1. S: 神デートEND（全体高評価）
2. A: いい感じEND
3. B: なんとか成立END
4. C: 気まずいEND
5. D: ドン引き爆笑END

## 16. 想定リスクと対策

1. リスク: ミニゲームが難しすぎる
対策: 成功判定幅を段階調整できるようにする

2. リスク: ネタが単調になる
対策: 失敗コメントと表情差分を重点追加

3. リスク: プレイ時間が伸びる
対策: 1交渉30秒固定とテンポ重視演出

## 17. 受け入れ条件（完了定義）

1. 1プレイで必ず7部位購入して終了できる
2. 各部位で成功またはランクダウン強制購入が機能する
3. 要望が毎回ランダムで1件選ばれる
4. 最終スコアに応じて5種のエンディング分岐が出る
5. 1周が平均8分以内で終わる

## 18. 開発メモ

1. 本仕様は初版。まずMVP実装後に数値調整を行う
2. 笑いの核は「チグハグ結果」と「ガンマの反応」
3. 失敗が価値になる設計を最優先する
